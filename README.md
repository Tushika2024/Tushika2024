## 💼 Projects

### 🔹 [CreditShield — AI Credit Risk & Limit Optimization](https://github.com/Tushika2024)
> End-to-end credit underwriting pipeline on 30K+ customer records

- ⚙️ Optuna-tuned XGBoost with reject inference and behavioral risk features
- 🔍 Explainable credit limit engine using SHAP and policy guardrails
- 📊 **ROC-AUC ≈ 0.82** | Streamlit-based decision dashboard for risk evaluation

`XGBoost` `Optuna` `SHAP` `Streamlit`

---

### 🔹 [SentinelLabel — Semantic Label Noise Detection (NLP)](https://github.com/Tushika2024)
> Pipeline for detecting mislabeled data across a 14K+ tweet dataset

- 🧬 Built with SentenceTransformers + UMAP + HDBSCAN
- ✅ Cross-encoder semantic validation
- 🚀 **3.6× precision improvement** in detecting mislabeled data

`SentenceTransformers` `UMAP` `HDBSCAN` `NLP`

---

### 🔹 [Sleep Behaviour Analysis for T2DM](https://github.com/Tushika2024)
> Wearable-driven analysis on the AI-READI dataset

- 🌙 Feature engineering across REM, variability, and sleep patterns
- 🤖 ML models validated with cross-validation
- 📊 **ROC-AUC ≈ 0.75** | **F1 ≈ 0.84**
- 📱 Built a Streamlit app for prediction & visualization

`Scikit-learn` `Feature Engineering` `Streamlit`
